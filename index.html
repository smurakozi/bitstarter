Crowdec

The site where the users design the applications.

Is that all that a software developer should know about the user?

We don't think so.
They should deliver the software the users really want.

On Crowdec you can ask people about your ideas so you can work on projects that have a real user interest.
Future users help you to pick the important features, and to recoginse those that just burn the time of the developers.
Real users give feedback about the existing features assisting in making them perfect.

Details
How does it work?
Anyone can create a project based on his/her idea or need.
The users can vote for projects they see promising so you can see how much interest your project has.
The owner or other users can come up with features and alternative ways of how the feature should wok.
Voting works for the features and the alternatives too, so developers know which features to implement first and how the users would like them to look and behave.
What's the difference compared to existing crowdfunding sites?
The main goal in Crowdec is not to get funding to implement your project, but to let the users shape it:
  • They select the best ideas
  • They help to decide which are the features worth developing
  • And they assist in designing it to really fulfill their needs
In addition to this you can ask for various kinds of contributions depending on your need: publicity (e.g. social sharing), help in the graphical design, coding, or whatever you need to implement your system.
Is it only for software developers?
Absolutely not.
You can start a project even if you are not able to realize it.
If you have a good idea, or a problem that is not (sufficiently) solved by any existing software that you know, create a project.
If it's good stuff you others will probably join and you will get help.
Is it only for software projects?
In the first turn we focus on software projects. In case of real user demand we will consider other areas.
In the mean time there is nothing preventing you from using it for any project that you like.
How can I help to make it happen?
First of all, help us to design a good site that fits to YOUR needs.
For any site using crowd intelligence it's important to have lots of users. Help us to get new users by sharing /following/liking us on social media. Refer to us in blogs, put a link pointing to our site or any specific project that you like on our site.
When and how can I use it?
As soon as the basic functions work reliably we would like to have real users. You will be able to browse the site without any ceremony. You can create your own project or contribute to others after a quick, free registration.
Like us on facebook and follow us on twitter and we keep you up to date about our progress.
What if I miss a feature or don't like something?
Use Crowdec :-).
We want to eat our own dogfood:
there will be a project for Crowdec where you can suggest features, alternatives and vote for the propsals of other users. Until the site goes live you can give us feedback via mail, or facebook.
